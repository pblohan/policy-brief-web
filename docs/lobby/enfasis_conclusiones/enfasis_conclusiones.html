Concepto y prop√≥sito del PCA

El PCA es una t√©cnica estad√≠stica que permite reducir la dimensionalidad de un conjunto de datos, manteniendo la mayor cantidad posible de variabilidad presente. Lo logra mediante la creaci√≥n de componentes principales ‚Äîcombinaciones lineales de las variables originales‚Äî que no est√°n correlacionados entre s√≠. Se utiliz√≥ para:

  • Identificar patrones subyacentes en datos complejos.
  • Resumir la informaci√≥n de m√∫ltiples variables sin perder la estructura.
  • Visualizar agrupamientos o trayectorias en datos de alta dimensi√≥n.
  • Facilitar an√°lisis posteriores (clustering, regresiones) con menor redundancia.

Interpretaci√≥n de componentes principales

Cada componente principal tiene::

  • Una proporci√≥n de varianza explicada (qu√© tanto resume del total de informaci√≥n).
  • Cargas que indican qu√© variables originales contribuyen m√°s.
  • PC1 (primer componente): Captura la mayor varianza total y est√° definido por las variables con m√°s peso.
  • (segundo componente): Es ortogonal (independiente) y recoge una dimensi√≥n secundaria pero relevante.

Cuando se grafican los puntos (en este caso, los a√±os) en el plano PC1-PC2, es posible observar agrupamientos, trayectorias y a√±os at√≠picos, enriqueciendo la lectura exploratoria con base estructurada.
